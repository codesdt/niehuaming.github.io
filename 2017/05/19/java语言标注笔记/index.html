标注

标注（ Annotation） 是 Java 语言 新出现 的 一个 特性， 在 实际 的 应用 中 其 可以 部分 或 全部 地 取代 传统 的 XML 等 部署 描述 文件。 之所以 要 出现 标注 特性， 是因为 部署 描述 文件 很复杂， 在 具体 编写 时 很容易 出错。

“@ SuppressWarnings”

其实“@ SuppressWarnings” 标注 就是 告诉 Java 编译器 不需要 再 提示“ 注意”， 即 起到 压缩 警告 的 作用。
