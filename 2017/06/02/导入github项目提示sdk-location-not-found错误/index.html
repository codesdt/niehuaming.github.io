导入github项目提示sdk location not found错误

发表于 2017-06-02 | 分类于 Adnroid | | 浏览

错误提示

当你要导入一个github项目的时候，会提示一个resolve error 错误，写的很明确就是没有定义默认的local.propertibs 文件的sdk路径，或则没有定义安卓环境变量，也就是安卓的home。

解决办法

  1. 暴力一点的解决办法就是关闭android studio 然后点击打开一个项目选择你的这个项目，就会自动的创建一个local.propertibs ,默认加上sdk.dir=/Users/nhm/Library/Android/sdk 这个目录，当然每一个环境不一样路径就会不一样。
  2. 配置一个ANDROID_HOME变量就行，写入export ANDROID_HOME=/Users/nhm/Library/Android/sdk。
